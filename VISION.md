# Telegram Mini App — BlackJack Casino

## Главная цель
Создать профессиональный Telegram Mini App с игрой BlackJack, 
где казино всегда имеет преимущество, включая защиту от счётчиков карт.

## Функциональность

### Игровой процесс
- Классический BlackJack с cut card (карта отреза)
- Cut card ставится случайно в последней трети колоды — счётчики карт не успевают использовать преимущество
- Несколько колод (6-8), перемешивание при достижении cut card
- Дилер бьёт до мягких 17
- Поддержка: Hit, Stand, Double Down, Split, Surrender
- BlackJack платит 3:2

### Сайд-беты
- Perfect Pairs — пара одинаковых карт
- 21+3 — покер-комбинация из 2 карт игрока + карта дилера
- Lucky Ladies — две дамы червей = джекпот
- Insurance — страховка при тузе дилера

### Мультиплеер
- До 7 игроков за одним столом одновременно
- Очередь ходов, таймер на ход (30 сек)
- Чат за столом
- Публичные и приватные столы

### Баланс и монетизация
- Стартовый баланс: 1000 фишек бесплатно
- Ежедневный бонус: 500 фишек
- Бесплатные игры: каждые 4 часа 3 бесплатных раунда
- Покупка фишек через Telegram Stars
- VIP статусы: Bronze/Silver/Gold/Platinum

### Защита от счётчиков карт
- Cut card в случайной позиции (60-75% колоды)
- Автоматическое обнаружение подозрительных паттернов ставок
- При обнаружении — принудительное перемешивание
- Лимиты ставок меняются динамически для подозрительных игроков

### Технический стек
- Frontend: React + Telegram Web App SDK + Framer Motion (анимации)
- Backend: Node.js + Express + WebSocket
- Database: SQLite (пользователи, балансы, история игр)
- Красивый UI: зелёное сукно, анимированные карты, звуки

## Структура проекта
- /src/game — игровой движок (колода, раздача, подсчёт очков, cut card)
- /src/server — API, WebSocket, мультиплеер
- /src/db — база данных, балансы, история
- /src/frontend — React компоненты, UI, анимации
- /src/anticheat — защита от счётчиков карт
- /test — тесты всей логики
